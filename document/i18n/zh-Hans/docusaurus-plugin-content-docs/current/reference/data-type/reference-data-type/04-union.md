# 联合类型（Union Type）

## 🐳类型简介 {#intro}

## 🏅语法示例 {#syntax-example}

```collie
string | number obj = 'aaa';
obj = 3.14;

string | character separator = ',';
[1, 2, 3].join(separator); // result: "1,2,3"
separator = " and ";
[1, 2, 3].join(separator); // result: "1 and 2 and 3"
```
