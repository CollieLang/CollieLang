# 贡献文档

本文档站点是使用 [Docusaurus](https://docusaurus.io/) 构建的。

### 安装

```sh
$ npm install
```

### 本地开发

```sh
$ npm run start
```

此命令将启动一个本地开发服务器，并打开一个浏览器窗口。大多数更改会实时反映，无需重启服务器。

### 构建

```sh
$ npm run build
```

此命令将生成静态内容到 `build` 目录中，可以使用任何静态内容托管服务进行部署。
